# Lalgudi Booth Finder - Setup Instructions

## Project Overview
This is a separate project dedicated to displaying polling booth information for Lalgudi Constituency only, containing 249 polling booths across Tiruchirappalli District.

## Quick Start

1. **Install dependencies**:
   ```bash
   bun install
   # or
   npm install
   ```

2. **Start development server**:
   ```bash
   bun run dev
   # or
   npm run dev
   ```

3. **Build for production**:
   ```bash
   bun run build
   # or
   npm run build
   ```

## Key Differences from Parent Project
- ✅ Lalgudi constituency only (no multi-constituency navigation)
- ✅ Simplified routing (single main page)
- ✅ 249 polling booth entries
- ✅ Standalone interactive map with booth details

## Technologies
- React + TypeScript
- Tailwind CSS
- Leaflet Maps
